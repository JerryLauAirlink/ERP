標準業務流程
客戶 → 報價單（Accepted）→ 工作 → 客戶 PO → 供應商採購單 Vendor PO → 應付（付供應商）→ 工作完成 → 應收（向客戶開立發票）

零、開始之前（管理員／Root）

  1. 確認右上角 地區（HK／TW／SG／MY／ID）。新增資料會標記在目前地區。
  2. 設定 → 一般：介面語言、時區、公司名稱。
  3. 設定 → 備份與還原：確認雲端金鑰／即時同步。重要操作後可按 立即同步。
  4. 設定 → 用戶管理：帳號、模組權限、地區權限。刪用戶須等到「已從雲端移除」。

一、主檔資料

1. 客戶

  • 客戶 → + 新增客戶；系統會建議下一個客戶編號。
  • Invoice Title、付款條件（30／45／60／90 Days 或自訂）、BU 可選填。

2. 供應商

  • 欄位對齊 Excel：編號、公司、GST、負責人、地址、郵遞區號、電話、聯絡人、電郵、手機、職稱、銀行、分行、帳號、SWIFT、信用條件。

3. 搬去其他地區

加錯地區時：開啟記錄 → 修改 → 表單最上方 「搬去其他地區」 → 選正確地區 → 儲存。
搬 Client 不會自動搬 Job／Invoice／PO，那些要各自改。

二、業務流程

步驟模組說明
①報價單選客戶；要連工作須 Accepted。Accepted 可自動開「新增工作」。
②工作選客戶、連報價（或勾「無報價單」自動編號如 HK00001）、填客戶 PO。詳情可見 BU、Vendor、Vendor PO。
③供應商採購單見下一節欄位說明。
④應付 AP連工作 → 選供應商 → 可填 AIRLINK PO／PO Amount。
⑤工作完成狀態 Completed。
⑥應收 AR連工作 → 向客戶開票。

詳情頁的藍色連結可直接開相關詳情（報價／工作／發票／Vendor PO／供應商）。

四、報表

  • 進行中清單：PO 餘額與開票進度。
  • 月報表：改資料後須按 同步月報。
  • 儀表板／總覽／管理層報表：依應收應付與已完成工作彙總。

五、Excel 匯入順序

  1. 客戶
  2. 供應商
  3. 報價單
  4. 工作
  5. 供應商採購單
  6. 應收
  7. 應付

匯入前確認地區；建議使用各模組的 Download Template。

工作匯入：填 Customer No 即可，Client Company 可留空（依客戶主檔自動對應）。匯出亦會帶 Customer No。

無正式報價：新增／修改工作 → 勾「無報價單」→ 依地區自動編號（HK00001／TW00001／SG00001…），儲存時建立 Accepted 佔位報價單。

六、常見錯誤

  • 未建客戶就建工作／報價
  • 報價未 Accepted 就連工作
  • 未建供應商就建 AP／Vendor PO
  • 地區選錯 → 用「搬去其他地區」，勿重複建檔
  • 改資料後未同步月報
  • 刪用戶後未「立即同步」→ 新瀏覽器可能還原
  • 未開雲端同步，只靠本機瀏覽器

七、檢查清單

  1. 地區正確；加錯已用「搬去其他地區」修正？
  2. 客戶／供應商主檔齊全？
  3. 報價 Accepted、工作／PO 完整？
  4. Vendor PO 與 AP 的 AIRLINK PO 已填？
  5. 月報與雲端同步正常？
Standard workflow
Client → Quotation (Accepted) → Job → Customer PO → Vendor PO → AP (pay vendor) → Job Completed → AR (invoice client)

0. Before you start (Admin / Root)

  1. Set the top-right Region (HK / TW / SG / MY / ID). New records are tagged to the current region.
  2. Settings → General: language, time zone, company name.
  3. Settings → Backup: confirm sync key / live sync. After critical changes, use Sync now.
  4. Settings → Users: accounts, module permissions, region access. After deleting a user, wait for “removed from cloud”.

1. Master data

1. Clients

  • Clients → + Add Client; next customer number is suggested.
  • Optional: Invoice Title, payment terms (30/45/60/90 Days or custom), BU.

2. Vendors

  • Columns match Excel: Vendor No, Company Name, GST #, Legal Rep, Address, Postal Code, Company Phone, Main Contact, E-mail, Mobile, Job Title, Bank Name, Branch, Account #, SWIFT, Credit Term.

3. Move to another region

If a record was created under the wrong region: open it → Edit → change Move to region at the top → Save.
Moving a Client does not auto-move related Jobs / Invoices / POs — edit those separately.

2. Transaction flow

1QuotationPick client; must be Accepted to link a job. Accepting can auto-open Add Job.
2JobClient, quotation(s) — or tick No quotation for auto nos like HK00001 — and customer PO lines. Detail shows BU, Vendor, Vendor PO.
3Vendor POSee column table below.

4. Reports

  • Ongoing List: PO balance and billing progress.
  • Monthly Report: after changes, click Sync monthly.
  • Dashboard / Overview / Management Report: AR/AP and completed-job margins.

5. Excel import order

  1. Clients
  2. Vendors
  3. Quotations
  4. Jobs
  5. Vendor PO
  6. AR
  7. AP

Confirm region first; use each module’s Download Template for headers.

Job import: fill Customer No only — Client Company is optional and auto-mapped. Export includes Customer No.

No formal quotation: on Add/Edit Job, tick No quotation to auto-assign HK00001 / TW00001 / SG00001… and create an Accepted placeholder on Save.

6. Common mistakes

  • Creating jobs/quotations before the client exists
  • Linking a job before quotation is Accepted
  • Creating AP / Vendor PO before the vendor master exists
  • Wrong region → use Move to region; do not duplicate the record
  • Forgetting to sync the monthly report
  • Deleting a user without Sync now → may reappear in a new browser
  • Relying only on local browser storage without live sync

7. Checklist

  1. Correct region (or moved with Move to region)?
  2. Client / vendor masters complete?
  3. Quotation Accepted; job / PO lines complete?
  4. Vendor PO and AP AIRLINK PO filled where needed?
  5. Monthly report + cloud sync healthy?
